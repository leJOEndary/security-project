Our project is based on native PHP We didn't use frameworks except for Jquery there's no needed installation you just need internet connection as we use Jquery library hosted on cloud

STEPS:
0- install XAMP (apache &Mysql servers) you can find it at 		https://www.apachefriends.org/index.html
1- Unzip the project folder.
2- go to htdocs in xamp directory default (C:\xampp\htdocs).
3- copy&paste project inside htdocs
4- Go to your browser locahost/phpmyadmin/
5- go to  import tab
6- choose the databasefolder from project files (tweety.sql) scroll down and click go.
7- go to your browser type localhost/TwitterS.

now it should be all set
